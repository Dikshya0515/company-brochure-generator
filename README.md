\# Company Brochure Generator



A simple Python tool that creates professional company brochures from website URLs using Ollama AI.



\## Features

\- Extracts real company information from websites

\- Finds contact details (email, phone, social media)

\- Generates professional PDF brochures

\- Uses Ollama for AI analysis



\## Requirements

```

requests

beautifulsoup4

fpdf

ollama-python

```



\## Usage

```python

from brochure\_generator import create\_company\_brochure

create\_company\_brochure("Tesla", "https://www.tesla.com")

```


\## Installation

1\. Install Ollama: https://ollama.ai

2\. Install Python packages:

```bash

pip install requests beautifulsoup4 fpdf ollama-python

```



3\. Run Ollama:

```bash

ollama serve

4\. Use the generator!
